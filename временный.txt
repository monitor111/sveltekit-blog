Есть бесплатные облачные MySQL-сервисы, которые можно использовать с Render, например:

Free tier у Planetscale (но там ограничения, и сейчас у вас с ним сложности)

Free tier у Remote MySQL — простой и бесплатный, но не для серьезных проектов

Free MySQL от freemysqlhosting.net — тоже простой вариант

Db4free.net — бесплатный тестовый MySQL сервер (нельзя использовать в проде)

Можно подключить базу с одного из этих сервисов к вашему проекту на Render.

Если хотите, могу помочь по шагам подключить базу на одном из таких сервисов.

Готовы?







Подключился к https://www.db4free.net/
Зайдите в ваш проект на Render — https://dashboard.render.com (войдите в аккаунт).

Перейдите в настройки вашего сервиса (вашего SvelteKit проекта).

Найдите раздел для переменных окружения (Environment Variables).

Проверьте, есть ли там переменные для подключения к базе, например:

DB_HOST — должен быть db4free.net

DB_USER — monitor98108

DB_PASSWORD — ваш пароль к базе на db4free.net

DB_NAME — sveltekit_blog3

DB_PORT — 3306 (стандартный MySQL порт)
